# ------------------------------
# 冒険者から受け取った冊子
# ------------------------------

:book-from-adv

■
きれいな本だ。
『路地裏に生まれ後に砂の海を渡り陸の孤島に流れ着いた放浪者の冒険記』と題字が書かれている。

■
\>【リル】
あれ？　この本、紙がつながっちゃってるね。
どうやって読むんだろう。

■
\>【ニエ】
これは冊子の小口をナイフで切るんですよ。
紙を折りたたんで製本するとこうなるので、最初に読む人が封切りをするんです。

■
\>【リル】
へー、詳しいね。
ニエはよく本を読むの？

■
\>【ニエ】
ええ、周囲には本が沢山ありましたので……。
ですが、実際に新品の冊子を見るのは初めてです。
とても綺麗でドキドキしますね。
